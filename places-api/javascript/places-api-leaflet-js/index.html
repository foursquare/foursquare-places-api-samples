Foursquare Places API Example with Leaflet

This example takes a query, near and category inputs to call Foursquare's Places API. The Places API returns Point of Interest (POI) data and populates the map on the page.

Check out Foursquare's Documentation for information on addtional inputs and uses of the Places API.

To use this example you must sign up for a developer account on foursquare.com and generate an API key and plug it in to the "Authorization" fields in main.js
